# Rosie Phạm Trading Official Website

The official professional platform for Rosie Phạm Trading, specialized in Forex, Gold (XAUUSD), and Crypto market analysis.

## Features

- **Real-time Trading Charts**: Integrated TradingView widgets with professional indicators (KingIchi Suite).
- **Zalo VIP Integration**: Live trade signals and community feed.
- **Economic Calendar**: Real-time USD and Forex news tracking.
- **Multi-Channel Chat**: Direct access via Zalo, WhatsApp, and Telegram.
- **SEO Optimized**: Fully optimized for search engines and social sharing.

## Tech Stack

- **Framework**: React with Vite
- **Styling**: Tailwind CSS & Shadcn UI
- **Animations**: Framer Motion & Swiper.js
- **Icons**: Lucide React
- **Data**: Custom scraped economic data & Zalo API integration

## Development

To run the project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the dev server:
   ```sh
   npm run dev
   ```

## Production Build

To build the project for production:

1. Run build command:
   ```sh
   npm run build
   ```
2. The output will be in the `/dist` directory.
